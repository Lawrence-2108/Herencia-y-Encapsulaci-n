# Tarea 3.1 - Herencia y Encapsulación

Proyecto en Java con tres clases:
- `AutoMovil.java`
- `HondaCivic.java`
- `Main.java`

## Estructura
- `AutoMovil`: clase base con 4 atributos privados, getters, setters y 4 métodos.
- `HondaCivic`: hereda de `AutoMovil` y sobrescribe 2 métodos.
- `Main`: prueba el funcionamiento de ambas clases.

## Cómo ejecutar
### En NetBeans
1. Abrir la carpeta del proyecto.
2. Ir a `src/main/java/tarea31/`.
3. Ejecutar la clase `Main.java`.

### En consola
Desde la raíz del proyecto:
```bash
javac -d out src/main/java/tarea31/*.java
java -cp out tarea31.Main
```
